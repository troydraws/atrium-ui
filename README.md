# Atrium UI
## Medium Grey

A nice UI theme for Atom.

This theme uses [Overpass](http://overpassfont.org/) as the default font.

The matching syntax UI theme is available [here](https://atom.io/packages/atrium-syntax). And it uses Overpass Mono.

# Atrium UI
## Medium Grey

A nice UI theme for Atom.

This theme uses [Overpass](http://overpassfont.org/) as the default font.

The syntax UI theme is available [here](https://github.com/troydraws/atrium-syntax). And it uses Overpass Mono.
